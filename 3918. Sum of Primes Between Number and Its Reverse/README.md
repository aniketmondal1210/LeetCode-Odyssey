# Position of the Only Set Bit

## Problem Statement

Given an integer `n`:

👉 Find the position of the **only set bit (1)** in its binary form  
👉 Position is **1-based from LSB (rightmost bit)**

---

## Conditions

- If **exactly one set bit exists** → return its position  
- Otherwise → return `-1`

---

# Examples

### Example 1
```
Input:  n = 2
Binary: 10
Output: 2
```

---

### Example 2
```
Input:  n = 5
Binary: 101
Output: -1
```

---

## Constraints:

- 1 <= n <= 1000
