# Exactly One Pair of Consecutive Set Bits

## Problem

Given an integer `n`, return:

- `true` if its binary representation contains **exactly one pair of consecutive set bits (11)**.
- `false` otherwise.

---

## Examples

### Example 1

**Input**

```text
n = 6
```

**Binary**

```text
110
```

There is exactly one occurrence of:

```text
11
```

**Output**

```text
true
```

---

### Example 2

**Input**

```text
n = 5
```

**Binary**

```text
101
```

There are no consecutive set bits.

**Output**

```text
false
```

---

## Constraints:

- 0 <= n <= 10^5
