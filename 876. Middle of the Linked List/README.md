# Middle of the Linked List

## Problem

Given the head of a singly linked list, return the **middle node** of the linked list.

If there are **two middle nodes**, return the **second middle** node.

---

## Examples

### Example 1

**Input**

```text
head = [1,2,3,4,5]
```

**Output**

```text
[3,4,5]
```

**Explanation**

The middle node is `3`, so return the list starting from node `3`.

---

### Example 2

**Input**

```text
head = [1,2,3,4,5,6]
```

**Output**

```text
[4,5,6]
```

**Explanation**

The list has two middle nodes (`3` and `4`).

According to the problem, return the **second middle** node (`4`).

---

## Constraints:

- The number of nodes in the list is in the range [1, 100].
- 1 <= Node.val <= 100
