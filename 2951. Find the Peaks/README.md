# Find Peaks in an Array

## Problem Statement

You are given an array `mountain`.

👉 A **peak** is an element that is:
```
Strictly greater than its neighbors
```

⚠️ Note:
- First and last elements **cannot** be peaks.

Return all peak indices in **any order**.

---

# Examples

### Example 1

**Input**
```
mountain = [2,4,4]
```

**Output**
```
[]
```

---

### Example 2

**Input**
```
mountain = [1,4,3,8,5]
```

**Output**
```
[1,3]
```

---

## Constraints:

- 3 <= mountain.length <= 100
- 1 <= mountain[i] <= 100
