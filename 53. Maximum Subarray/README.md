# Maximum Subarray Sum

## Problem Statement
You are given an integer array `nums`.

Your task is to find the **contiguous subarray** (containing at least one number) which has the **largest sum**, and return that sum.

---

## Examples

### Example 1
**Input:**

nums = [-2,1,-3,4,-1,2,1,-5,4]


**Output:**

6


**Explanation:**  
The subarray `[4, -1, 2, 1]` has the largest sum:

4 + (-1) + 2 + 1 = 6


---

### Example 2
**Input:**

nums = [1]


**Output:**

1


**Explanation:**  
The only subarray `[1]` has the maximum sum.

---

### Example 3
**Input:**

nums = [5,4,-1,7,8]


**Output:**

23


**Explanation:**  
The subarray `[5, 4, -1, 7, 8]` has the largest sum:

5 + 4 - 1 + 7 + 8 = 23


---

## Constraints

- 1 <= nums.length <= 10^5
- -10^4 <= nums[i] <= 10^4
