# Remove All Duplicates from Sorted Linked List II

## Problem

Given the head of a **sorted linked list**, delete all nodes that have duplicate values, leaving **only distinct numbers** from the original list.

Return the modified linked list.

---

## Examples

### Example 1

**Input**

```text
head = [1,2,3,3,4,4,5]
```

**Output**

```text
[1,2,5]
```

---

### Example 2

**Input**

```text
head = [1,1,1,2,3]
```

**Output**

```text
[2,3]
```

---

## Constraints:

- The number of nodes in the list is in the range [0, 300].
- -100 <= Node.val <= 100
- The list is guaranteed to be sorted in ascending order.
