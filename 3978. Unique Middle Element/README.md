# Check if the Middle Element Appears Exactly Once

## Problem

You are given an integer array `nums` of **odd** length `n`.

Return **true** if the middle element of `nums` appears **exactly once** in the array. Otherwise, return **false**.

---

## Examples

### Example 1

**Input**

```text
nums = [1,2,3]
```

**Output**

```text
true
```

**Explanation**

The middle element is `2`, and it appears exactly once in the array.

---

### Example 2

**Input**

```text
nums = [1,2,2]
```

**Output**

```text
false
```

**Explanation**

The middle element is `2`, and it appears twice in the array.

---

## Constraints:

- 1 <= n == nums.length <= 100
- n is odd.
- 1 <= nums[i] <= 100
