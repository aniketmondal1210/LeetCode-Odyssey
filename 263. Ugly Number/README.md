# Ugly Number

## Problem Statement
An **ugly number** is a positive integer whose prime factors include **only** `2`, `3`, and `5`.

Given an integer `n`, return **true** if `n` is an ugly number, otherwise return **false**.

---

## Examples

### Example 1
**Input:**  

n = 6

**Output:**  

true

**Explanation:**  
`6 = 2 × 3`, which uses only allowed prime factors.

---

### Example 2
**Input:**  

n = 1

**Output:**  

true

**Explanation:**  
`1` has no prime factors and is considered an ugly number.

---

### Example 3
**Input:**  

n = 14

**Output:**  

false

**Explanation:**  
`14 = 2 × 7` includes the prime factor `7`, which is not allowed.

---

## Constraints

-2³¹ ≤ n ≤ 2³¹ - 1
