# Convert Integer to Base 7

## Problem Statement
Given an integer `num`, return a string of its **base 7 representation**.

---

## Examples

### Example 1
**Input:**

num = 100

**Output:**

"202"


### Example 2
**Input:**

num = -7

**Output:**

"-10"


---

## Constraints
- -10⁷ ≤ num ≤ 10⁷
