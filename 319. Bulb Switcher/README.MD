# Bulb Switcher

## Problem Statement

There are `n` bulbs initially turned **off**.

You perform `n` rounds:

1. Turn on all bulbs.
2. Turn off every 2nd bulb.
3. Toggle every 3rd bulb.
4. Toggle every 4th bulb.
...
n. Toggle only the nth bulb.

Return the number of bulbs that remain **on** after all rounds.
