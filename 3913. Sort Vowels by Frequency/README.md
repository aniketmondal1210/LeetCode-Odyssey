# Rearrange Vowels by Frequency

## Problem Statement

Given a string `s`, rearrange **only the vowels** such that:

1. Vowels are sorted in **non-increasing order of frequency**
2. If frequencies are equal → maintain **first occurrence order**

👉 Consonants remain at the same position  
👉 Return the modified string  

---

# Examples

### Example 1

**Input**
```
s = "leetcode"
```

**Output**
```
"leetcedo"
```

---

### Example 2

**Input**
```
s = "aeiaaioooa"
```

**Output**
```
"aaaaoooiie"
```

---

### Example 3

**Input**
```
s = "baeiou"
```

**Output**
```
"baeiou"
```

---

## Constraints:

- 1 <= s.length <= 10^5
- s consists of lowercase English letters
