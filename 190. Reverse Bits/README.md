# Reverse Bits of a 32-Bit Signed Integer

## Problem Statement
Given a **32-bit signed integer `n`**, reverse its bits and return the resulting integer.

---

## Examples

### Example 1
**Input:**

n = 43261596


**Binary:**

00000010100101000001111010011100


**Output:**

964176192


**Binary (reversed):**

00111001011110000010100101000000


---

### Example 2
**Input:**

n = 2147483644


**Binary:**

01111111111111111111111111111100


**Output:**

1073741822


**Binary (reversed):**

00111111111111111111111111111110


---

## Constraints:

- 0 <= n <= 231 - 2
- n is even.
