# Find First and Last Position of Element in Sorted Array

## Problem Statement

Given a sorted array `nums`:

👉 Find the **starting and ending index** of a given `target`

👉 If target not found:
```
return [-1, -1]
```

👉 Required complexity:
```
O(log n)
```

---

# Examples

### Example 1
```
Input:  nums = [5,7,7,8,8,10], target = 8
Output: [3,4]
```

---

### Example 2
```
Input:  nums = [5,7,7,8,8,10], target = 6
Output: [-1,-1]
```

---

### Example 3
```
Input:  nums = [], target = 0
Output: [-1,-1]
```

---

## Constraints:

- 0 <= nums.length <= 10^5
- -10^9 <= nums[i] <= 10^9
- nums is a non-decreasing array.
- -10^9 <= target <= 10^9
