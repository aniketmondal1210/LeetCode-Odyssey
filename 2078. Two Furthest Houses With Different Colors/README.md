# Maximum Distance Between Houses with Different Colors

## Problem Statement

You are given an array `colors` where:

```
colors[i] = color of ith house
```

👉 Find the **maximum distance** between two houses having **different colors**.

```
distance = |i - j|
```

---

# Examples

### Example 1

**Input**
```
colors = [1,1,1,6,1,1,1]
```

**Output**
```
3
```

---

### Example 2

**Input**
```
colors = [1,8,3,8,3]
```

**Output**
```
4
```

---

### Example 3

**Input**
```
colors = [0,1]
```

**Output**
```
1
```

---

## Constraints:

- n == colors.length
- 2 <= n <= 100
- 0 <= colors[i] <= 100
- Test data are generated such that at least two houses have different colors.
