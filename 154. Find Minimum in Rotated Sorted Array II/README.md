# Find Minimum in Rotated Sorted Array (With Duplicates)

## Problem Statement

Given a sorted array that is **rotated** and may contain **duplicates**:

👉 Find the **minimum element**

---

# Examples

### Example 1
```
Input:  [1,3,5]
Output: 1
```

---

### Example 2
```
Input:  [2,2,2,0,1]
Output: 0
```

---

## Constraints:

- n == nums.length
- 1 <= n <= 5000
- -5000 <= nums[i] <= 5000
- nums is sorted and rotated between 1 and n times.
