# Convert Array to 2D Array with Minimum Rows

## Problem Statement

Given an array `nums`:

👉 Create a 2D array such that:
- Each row has **distinct elements**
- All elements from `nums` are used
- Number of rows is **minimum**

---

# Examples

### Example 1
```
Input:  [1,3,4,1,2,3,1]

Output:
[[1,3,4,2],
 [1,3],
 [1]]
```

---

### Example 2
```
Input:  [1,2,3,4]

Output:
[[4,3,2,1]]
```

---

## Constraints:

- 1 <= nums.length <= 200
- 1 <= nums[i] <= nums.length
