# Big Countries

## Problem Statement

Find the countries that are considered **big**.

A country is big if:

- `area >= 3000000`
OR
- `population >= 25000000`

Return:

- `name`
- `population`
- `area`

---

# Example

## Input

| name        | area    | population |
|--------------|----------|-------------|
| Afghanistan | 652230  | 25500100 |
| Albania     | 28748   | 2831741 |
| Algeria     | 2381741 | 37100000 |

---

## Output

| name        | population | area    |
|--------------|-------------|----------|
| Afghanistan | 25500100 | 652230 |
| Algeria     | 37100000 | 2381741 |
