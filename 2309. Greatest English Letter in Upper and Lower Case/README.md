# Greatest Letter in Both Uppercase and Lowercase

## Problem Statement

Given a string `s`, find the **greatest English letter** that appears in **both lowercase and uppercase**.

👉 Return the letter in **UPPERCASE**  
👉 If none exists, return `""`

---

# Examples

### Example 1

**Input**
```
s = "lEeTcOdE"
```

**Output**
```
"E"
```

---

### Example 2

**Input**
```
s = "arRAzFif"
```

**Output**
```
"R"
```

---

### Example 3

**Input**
```
s = "AbCdEfGhIjK"
```

**Output**
```
""
```

---

## Constraints:

- 1 <= s.length <= 1000
- s consists of lowercase and uppercase English letters.
