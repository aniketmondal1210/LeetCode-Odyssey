# Ugly Number II

## Problem Statement
An **ugly number** is a positive integer whose **prime factors are limited to 2, 3, and 5**.

Given an integer `n`, return the **n-th ugly number**.

By convention, **1 is considered the first ugly number**.

---

## Examples

### Example 1
**Input:**  

n = 10


**Output:**  

12


**Explanation:**  
The first 10 ugly numbers are:

[1, 2, 3, 4, 5, 6, 8, 9, 10, 12]

The 10th ugly number is `12`.

---

### Example 2
**Input:**  

n = 1


**Output:**  

1


**Explanation:**  
`1` has no prime factors, so it is considered an ugly number.

---

## Constraints

1 ≤ n ≤ 1690
