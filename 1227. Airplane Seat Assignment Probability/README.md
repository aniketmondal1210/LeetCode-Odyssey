# Airplane Seat Probability Problem

## Problem Statement

There are `n` passengers and exactly `n` seats.

- The **first passenger** lost their ticket and chooses a seat randomly.
- Every other passenger:
  - Takes their own seat if available.
  - Otherwise, picks a random free seat.

Return the probability that the **nth passenger** gets their own seat.
